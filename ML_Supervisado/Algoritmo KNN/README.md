
# Proyecto de Algoritmo KNN

Este proyecto implementa el algoritmo K-Nearest Neighbors (KNN) utilizando Python y la biblioteca `pandas` para el análisis de datos. El objetivo principal es analizar un conjunto de datos relacionado con recursos humanos y aplicar el algoritmo KNN para resolver problemas específicos.

</p>


## Archivos del Proyecto

- **Tarea M21-CD – RaulCortesAmador.ipynb**: Contiene el código principal del proyecto, incluyendo la carga y análisis del conjunto de datos, así como la implementación del algoritmo KNN.
- **recursos_humanos.csv**: Archivo CSV que contiene los datos relacionados con recursos humanos. Este archivo debe estar en el mismo directorio que el notebook para que el código funcione correctamente.

## Requisitos

Asegúrate de tener instaladas las siguientes dependencias antes de ejecutar el proyecto:

- Python 3.x
- Pandas
- Jupyter Notebook

Puedes instalar las dependencias necesarias ejecutando:

```bash
pip install pandas notebook
```

## Uso

1. Clona este repositorio en tu máquina local.
2. Asegúrate de que el archivo `recursos_humanos.csv` esté en el mismo directorio que el notebook.
3. Abre el archivo `Tarea M21-CD – RaulCortesAmador.ipynb` en Jupyter Notebook.
4. Ejecuta las celdas del notebook para cargar y analizar los datos, y para aplicar el algoritmo KNN.

## Manejo de Errores

El código incluye manejo de errores para garantizar una ejecución fluida:

- Si el archivo `recursos_humanos.csv` no se encuentra en el directorio, se mostrará el mensaje:  
  ```
  Error: 'recursos_humanos.csv' not found. Please ensure the file is in the correct directory or provide the full path.
  ```
- Si ocurre algún otro error, se mostrará un mensaje con la descripción del error.
