# Estimación del PIB de México en 2022 utilizando Regresión Lineal

Este proyecto utiliza técnicas de Machine Learning, específicamente **Regresión Lineal**, para estimar el Producto Interno Bruto (PIB) de México en el año 2022. A través del análisis de datos históricos, se busca construir un modelo predictivo que permita realizar estimaciones precisas del PIB.

## Descripción del Proyecto

El objetivo principal de este proyecto es analizar datos históricos del PIB de México y aplicar un modelo de regresión lineal para predecir el valor del PIB en el año 2022. Este proyecto es una introducción práctica al uso de algoritmos de Machine Learning en el análisis de datos económicos.

## Conjunto de Datos

El conjunto de datos utilizado contiene información histórica del PIB de México, incluyendo:

- **Periodo**: Años en los que se registraron los datos.
- **GDP (Producto Interno Bruto)**: Valores históricos del PIB de México.

Los datos han sido recopilados de fuentes confiables y representan un panorama general del crecimiento económico de México en los últimos años.

## Análisis de Datos

El análisis de datos incluye:

1. **Visualización de los datos históricos**: Se graficaron los datos para observar tendencias y patrones en el crecimiento del PIB.
2. **Preprocesamiento de datos**: Se limpiaron y prepararon los datos para su uso en el modelo de regresión lineal.
3. **Entrenamiento del modelo**: Se utilizó un modelo de regresión lineal para ajustar los datos históricos y generar una ecuación que relacione el periodo con el PIB.
4. **Evaluación del modelo**: Se evaluó el desempeño del modelo utilizando métricas como el error cuadrático medio (MSE) y el coeficiente de determinación (R²).

## Conclusiones

- El modelo de regresión lineal mostró ser una herramienta útil para estimar el PIB de México basado en datos históricos.
- La predicción para el PIB de México en el año 2022 se realizó con un nivel aceptable de precisión, aunque se identificaron limitaciones debido a posibles factores externos no considerados en el modelo.
- Este proyecto demuestra cómo las técnicas de Machine Learning pueden ser aplicadas en el análisis económico para realizar predicciones y tomar decisiones informadas.

2. Instala las dependencias necesarias utilizando `pip install -r requirements.txt`.
